# apidoc-practice-demo
A quick demo using apiDoc to create api documents.

> initial commit on 26 Jan 2018

### how to operate

1. `npm install`
install dependencies

2. `npm run test`

apiDoc HTML files will be created by script `apidoc -i src/ -o out`  
  
running:  
server: `localhost` 
server port: `8080`
